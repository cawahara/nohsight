# Nohsight.com
Mobile Web App for organize Noh events (and search them)

## Requirements
- PostgreSQL

## Get started
```
$ git clone https://github.com/Shantti-Y/Noh-event
$ cd Noh-app
$ bundle install --path vendor/bundle
$ yarn install
```
## Database & Data
```
$ bin/rails db:create
$ bin/rails db:migrate
$ bin/rails db:seed_fu
```

##  Run Rails
```
$ bin/rails server
```

## Test

```
$ rspec
```
